# My Claude Code Skills

这个仓库保存我安装的所有 [Claude Code](https://claude.ai/code) 技能。

## 已安装技能

| 技能 | 来源 | 描述 |
|------|------|------|
| find-skills | [vercel-labs/skills](https://github.com/vercel-labs/skills) | 帮助发现和安装 agent 技能 |

## 如何安装技能

```bash
npx skills add <owner/repo> --skill <skill-name> -g -y
```

安装后自动出现在此仓库中，提交并推送即可备份。
